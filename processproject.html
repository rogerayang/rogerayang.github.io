Project 4| Processing Project

Using Dan Shiffman's bouncing ball processing code formula as the basics for the project a fundamental requirement for the assignment was that each student created their own version of the bouncing ball processing animation.
Removing the background and using a randomization code along with a combination of "if" and "If else" processing codes I created a psychodelic effect further enhanced by a processing code that altered the perspective of scale on a given object with unexpected but with excellently animated results.
